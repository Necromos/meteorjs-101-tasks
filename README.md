#MeteorJS 101: Tasks lists

###Meteor packages managment

```sh
meteor add bootstrap
meteor add jquery
meteor add account-ui
meteor add accounts-facebook
meteor remove insecure
meteor remove autopublish
```

###Application structure

```
model.js
public/
client/
		tasks.js
		tasks.html
		tasks.css
server/
		server.js		
```

###Run application

```sh
meteor
````
